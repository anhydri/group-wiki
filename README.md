# MCCG
Repository for the Monash computational chemistry group

# Comp_chem_packages

Includes input files and job scripts for GAMESS, GAUSSIAN and Psi4
computational chemistry packages, along with job scripts for the Magnus,
Massive, Monarch, Raijin and Stampede2 (Texas) supercomputers. 

# Scripts

Useful scripts that help with input file generation and visualisation of results. 
 
# Tutorials

Covers topics such as programming and running commands involving job submission
using the
[PBS](https://www.pbsworks.com/PBSProduct.aspx?n=Altair-PBS-Professional&c=Overview-and-Capabilities)
and [SLURM](https://slurm.schedmd.com/) job schedulers.
